## Second Site – RateMyVSO.com (Vision)

**Purpose:** “Yelp for Veterans Service Officers” — veterans review their VSOs so others can find accredited, effective help.

### Core features (current vision)

- Search VSOs by location
- Star rating + written reviews + comments
- Individual VSO profile pages  
  Example: *John Smith – DAV – Virginia Beach ⭐⭐⭐⭐⭐ “Helped me get 90% on my first claim.”*

### Enhanced idea – Verified VSO directory + reviews

- **Profile fields:** Name, organization (DAV, VFW, American Legion, State VSO, etc.), city/state, star rating, veteran reviews, link to **verify accreditation** (VA link above)
- **Actions:** Leave a Review • Verify Accreditation • Contact Organization
- **Sections:** Top Rated VSOs Near You • Veteran Success Stories • Searchable directory by city/state  
  Example: *Top Rated VSOs in Virginia Beach: 1. DAV – Norfolk ⭐⭐⭐⭐⭐ 2. Virginia Dept of Veterans Services ⭐⭐⭐⭐ …*
- **Compliance:** Platform does not assist with filing claims; it helps veterans find accredited reps and share experiences per VA guidelines.
